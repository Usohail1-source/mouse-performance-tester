# 🖱️ MouseTracer — Mouse Performance Tester

MouseTracer is a web-based mouse skill testing platform designed to measure key aspects of cursor performance through interactive test modes. It focuses on real user input metrics such as reaction speed, tracking control, precision, and click speed.

---

## 🚀 Live Overview

This project simulates a lightweight performance lab where users can test and improve their mouse skills in a clean, responsive environment.

---

## 🎯 Features

### 1. Reaction Test

* Measures how quickly a user responds to visual changes
* Displays reaction time in milliseconds
* Tracks best and average performance

---

### 2. Mouse Tracking Test

* Follow a moving target using **hover only (no clicking)**
* Tests:

  * Cursor control
  * Tracking precision
  * Real-time movement accuracy
* Dynamic difficulty and moving targets create a more realistic “aim training” feel

---

### 3. Click Speed Test (CPS)

* Measures clicks per second over a timed session
* Includes:

  * Live CPS tracking
  * Total clicks
  * Peak CPS
* Pre-game countdown allows users to position their mouse before starting

---

## 🎮 How to Use

1. Open the website
2. Select a test mode from the home screen
3. Complete the test
4. View your performance metrics
5. Retry to improve your score
