# 🎯 REFLEX — Mouse Skill Tester

A browser-based web app designed to test and improve **reaction time** and **mouse control** through fast-paced, skill-based mini games.

---

## 🚀 Overview

REFLEX is an interactive training tool that challenges users in two core areas:

* ⚡ **Reaction Speed**
* 🎯 **Cursor Precision & Aim**

Built with pure **HTML, CSS, and JavaScript**, the app runs entirely in the browser with no external dependencies.

---

## 🎮 Game Modes

### ⚡ Reaction Test

* Wait for the screen to turn green
* Click as fast as possible
* Measures reaction time in milliseconds (ms)
* Tracks:

  * Best time
  * Average time
  * Total attempts

---

### 🎯 Aim Trainer

* Targets appear across the screen
* Move your mouse over targets to hit them
* Difficulty increases over time:

  * More targets
  * Faster pacing
* Game ends after 3 misses

Tracks:

* Score
* Level reached
* Average capture speed (ms)
* Best streak

---

## 🧠 Features

* Real-time mouse tracking
* Progressive difficulty scaling
* Smooth animations and visual feedback
* Sound effects for interaction
* Persistent stats using local storage
* Clean UI with a neon arcade theme

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (custom styling + animations)
* Vanilla JavaScript (no frameworks)
* Canvas API for rendering game elements
